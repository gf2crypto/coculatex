Шабон статьи для журнала "Дискретная математика"
Используется pdflatex
Параметры:
   jobname: <file_name>  # name of produced pdf file
   lang: 'ru' | 'en'  # язык документа
   title: <str>  # название статьи
   authors:
    -
        name: <str>  # имя
        institute: <str>  # место работы
        email: <str>
    -
        ....
   udk: <str>  # УДК
   support: <str>  # информация о том, кто поддержала работу, либо
                   # номера гранта РФФИ или другой организации
                   # Пример:
                   #   Работа выполнена при поддержке РФФИ, номер гранта 112339912
   keywords: <str>  # ключевые слова
   abstract: <str>  # аннотация
